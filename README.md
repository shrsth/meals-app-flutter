# Meals App (Flutter)

A Flutter-based meals application built to practice and understand core Flutter concepts such as multi-screen navigation, state handling, and widget-based UI architecture.

## Overview
This app allows users to browse meals by category, view detailed meal information, apply dietary filters, and manage a list of favorite meals. The project focuses on building a structured, multi-screen Flutter application rather than backend or production deployment.

## Features
- Category-wise meal browsing
- Meal detail view with ingredients and steps
- Dietary filters (vegetarian, vegan, gluten-free, lactose-free)
- Favorites management
- Tab and drawer-based navigation
- Clean and reusable widget structure

## Tech Stack
- Flutter (Dart)
- Material UI
- Local in-memory data handling
- Git & GitHub

## State Management
- Widget-based state management using StatefulWidgets
- Lifted state and callbacks for sharing data across screens

## Platform Support
- Android
- iOS

## Status
Learning project focused on Flutter navigation, UI architecture, and state flow.  
Planned improvements include backend integration and scalable state management.
